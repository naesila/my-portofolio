## 🎯 Cara Download

### Opsi 1: Download dari Figma Make (TERCEPAT)
1. Di Figma Make, cari tombol **"Download"** atau **"Export"**
2. Download sebagai ZIP file
3. Extract di komputer Anda
4. Buka folder dengan VS Code
5. Done! ✅

### Opsi 2: Copy Manual dari Browser
1. Klik kanan pada preview Figma Make
2. Pilih **"Inspect"** atau **"View Source"**
3. Download semua file yang diperlukan

### Opsi 3: Clone/Download dari Git (jika sudah di-push)
```bash
git clone <repository-url>
cd my-portfolio
npm install
npm run dev
```
